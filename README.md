Mariso Production

Full-stack e-commerce platform for handcrafted candles and home décor.

Tech Stack
	•	Frontend: React (CRACO)
	•	Backend: FastAPI
	•	Database: MongoDB (assumed)

Setup

Frontend

cd frontend
npm install
npm run dev

Backend

cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app –reload

Features
	•	Product listing
	•	Cart & checkout
	•	Admin panel
	•	Order management
